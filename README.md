######### About the Project    #########
The Smart Bank Management System is a full-stack web application built using Java, Spring MVC, JPA, JSP, and MySQL. It provides secure banking features such as OTP-based login, role-based access for customers and admins, fund transfers, mini-statements, and loan management.

The system includes:
- Secure OTP login and session handling  
- Customer dashboard with real-time balance and transaction history  
- Loan application and status tracking  
- Admin panel for customer management and reporting  
- Optimized MySQL schema for accurate and efficient transactions  
- Deployment on Apache Tomcat
